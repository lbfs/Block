# Block
A game in which rotatable pieces fall down a board.

[![Screenshot](./screenshot.png)](https://www.youtube.com/watch?v=H8BKhYOtPWM)

# Controls
S - Push Block Down  
A - Push Block Left  
D - Push Block Right  
J - Rotate Block  
SPACE - Rapid Drop Block
